从空白工作区开始

程序是本地题库编辑器，不是在线上传服务。你可以不联网完成导入、整理、复制和大部分导出。

1. 安装或直接运行

安装版写入当前用户目录；便携版解压后直接运行。两者都将用户数据保存在 %LOCALAPPDATA%\AnewWorkbench，卸载默认不删除工作区。

安装版便携版

2. 拖拽到暂存区

支持正文与 Supporting Information PDF、CSV、TSV、XLSX 数据集，以及 Markdown、TXT、DOCX、PDF、CSV、XLSX 题目文件。导入不会直接覆盖工作区。

3. 预览并确认

检查文件分类、题目拆分和字段映射后再确认。导入错误时可取消暂存；已确认内容可软删除并从回收站恢复。

4. 复制与导出

可复制单题或整套题目，并导出 Markdown、XLSX、CSV ZIP、学生版或专家版 PDF、原始资源与完整 .anewpack。

5. 核对下载

从 GitHub Releases 下载 SHA256SUMS.txt，使用 Windows PowerShell 的 Get-FileHash 文件名 -Algorithm SHA256 与清单比对。
