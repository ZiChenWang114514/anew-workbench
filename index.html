LOCAL-FIRST RESEARCH WORKBENCH

把论文、数据表与题目，整理成一个可迁移的本地工作区。

拖入正文、Supporting Information、CSV、XLSX 与题目文件，先预览、再确认； 在本机完成编辑、恢复、复制和多格式导出。

下载 Windows 安装版 下载便携版

Windows 10/11 x64 · 无需预装 Python 或 Node · 用户工作区默认保留在本机

01

先暂存，后入库

拖拽导入 PDF、CSV、TSV、XLSX、Markdown、TXT、DOCX 与 .anewpack；分类和字段映射均可在确认前调整。

02

可恢复的编辑

题目和资源采用软删除，支持即时撤销、回收站恢复与历史版本回退，减少误操作成本。

03

面向科研数据

保留 SMILES、检测边界、长数字和原始文件字节；导出 Markdown、XLSX、CSV ZIP、PDF 与 .anewpack。

PRIVATE BY DEFAULT

你的论文和数据不需要进入公开网站。

GitHub Pages 只承载产品说明、完全合成的只读演示和下载入口。实际工作区由本地伴侣程序管理。

智能拆题是可选功能，只有用户明确点击提交的题目文本才会发送；不会上传正文或 Supporting Information。
